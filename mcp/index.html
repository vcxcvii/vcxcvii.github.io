This site supports the Model Context Protocol (MCP) — a standard that lets AI agents read and search websites directly. If you use Claude Desktop, Cursor, or another MCP-compatible client, you can connect it here and ask your agent questions about anything I've written.

What is this?

MCP is an open standard that lets AI assistants access external tools and data sources — including websites. Once you connect your agent to this site, it can browse my writing, look up specific posts, and search across all my notes on B2B marketing, GTM strategy, sales enablement, and career — without you having to copy-paste anything.

For example, once connected you could ask your agent:

  • "What has Varun written about imposter syndrome?"
  • "Summarise his notes on GTM strategy"
  • "Find posts tagged 'career' and pull the key takeaways"

Available tools

Four tools are exposed to your agent once connected:

  • list_posts — all notes with titles, dates, and tags
  • get_post — full text of any note by slug
  • search_posts — keyword search across all notes
  • get_site_info — about me, topics I cover, and links

How to connect

You need an MCP-compatible client (Claude Desktop or Cursor work well). Setup takes about two minutes.

  1. Add the WebMCP bridge to your MCP client
    In Claude Desktop: Settings → Developer → Edit Config, then add: 
    {
  "mcpServers": {
    "webmcp": {
      "command": "npx",
      "args": ["-y", "@jason.today/webmcp@latest", "--mcp"]
    }
  }
}
    Restart Claude Desktop after saving.
  2. Generate a token
    Ask your agent: "Make a webmcp token for varunchoraria.com"
  3. Paste the token here
    Click the blue widget in the bottom-right corner of this page and paste your token. Your agent can now read this site.

Why MCP?

Most AI assistants have a knowledge cutoff and don't know what I've written recently. MCP solves this by giving your agent live, direct access — no RAG pipeline, no copy-pasting, no stale embeddings. If you're researching B2B marketing or GTM strategy and already use an AI agent in your workflow, this is the fastest way to pull in my writing as context.
