This site exposes a live MCP server — add one URL to your AI client and your agent can read, search, and reference everything I've written. No plugins, no copy-pasting.

Connect in 30 seconds

Pick your client:

Run this in your terminal:

claude mcp add --transport http varunchoraria https://varunchoraria-mcp.vercel.app

That's it. Restart Claude Code and ask it anything about my writing.

Open Settings → Developer → Edit Config and add:

{
  "mcpServers": {
    "varunchoraria": {
      "type": "http",
      "url": "https://varunchoraria-mcp.vercel.app"
    }
  }
}

Restart Claude Desktop after saving.

Go to Settings → MCP and add a new server, or edit .cursor/mcp.json:

{
  "mcpServers": {
    "varunchoraria": {
      "type": "http",
      "url": "https://varunchoraria-mcp.vercel.app"
    }
  }
}

Any MCP-compatible client that supports HTTP transport works. Point it at:

https://varunchoraria-mcp.vercel.app

Transport: Streamable HTTP (MCP spec 2024-11-05).

What you can ask

Once connected, try:

  • "What has Varun written about imposter syndrome?"
  • "Summarise his notes on GTM strategy."
  • "Find posts tagged 'career' and pull the key takeaways."
  • "What's his most recent post about?"

Available tools

  • get_site_info — background, topics, contact links
  • list_posts — all notes with titles, dates, and tags
  • get_post — full text of any note by slug
  • search_posts — keyword search across all notes

Why MCP?

Most AI assistants have a knowledge cutoff and don't index personal sites. MCP gives your agent live access — no RAG pipeline, no copy-pasting. If you're doing research on B2B marketing or GTM strategy and already use an AI agent, this is the fastest way to pull my writing in as context.
